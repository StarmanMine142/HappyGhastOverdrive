Tired of your happy ghast being slow? This mod introduces a new enchantment "Aerial Swiftness."

___     
### **Characteristics**
- Maximum level: 3
- Average speed happy ghast: 14~15 b/s
- Maximum speed: 17.5 b/s
___
### **How to get**
- In the bastion's treasuries
- Trading with piglins
